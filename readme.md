4. #### 用户分层

   ##### <u>4.1 RFM模型</u>

   > R（Recency）：最近一次消费时间。反映了用户最近消费的热度，用以衡量用户是否流失。理论上，最近一次消费时间越长，流失概率越高
   >
   > 
   >
   > F（Frequency）：用户的消费频率。反映了用户对于产品、品牌的忠诚程度。理论上，一定时间内的购买频率越高，用户忠诚度越高
   >
   > 
   >
   > M（Monetary）：消费金额。反映了用户的购买力。
